  • 我怎样将论坛的议程添加至日历应用？

    您可以在日历应用中订阅我们的日程（即将日程添加至您的日历应用）。我们的日历源如下：

    {% include ical-box.html load_js="true" %}

    由于不同设备的操作系统各异，您可能需要按照不同的步骤操作。

    • 在Mac上使用，您可以参阅此帮助信息。
    • 在iOS设备上使用，您可以参阅此帮助信息。
    • 在Android设备上，由于并非所有系统都设置有默认的日历应用，您可以利用此应用订阅我们的在线日历。
    • 在Windows设备上，您可以在microsoft账户中订阅我们的在线日历，并利用Office Outlook或Windows Calendar等应用同步至您的设备。

  • 我怎样在移动设备上使用论坛网站？

    我们的整个网站都针对移动设备进行了优化，因此您可以随时在您的网络浏览器中访问我们的网站。为了确保最佳使用效果，我们建议您将浏览器应用更新至最新版本。

    另外，您还可以在您的移动设备上像使用原生应用一样使用我们的网站，您可以参阅我们的PWA页面了解如何操作。

  • 为什么有些链接带有标示？

    带有标示的链接指向第三方提供的内容，我们对这些内容无法进行监督或控制，因此希望通过此标示进行提示，如有必要，请在使用这些链接时额外注意。

  • 我怎样使用RSS源？

    RSS（Really Simple Syndication）是一种在您最喜欢的设备上阅读新闻及动态资讯的便捷方式。我们博客文章的RSS源如下：

    {% include rss-box.html %}

    根据不同设备的操作系统，有许多RSS阅读器可以选择使用。

    • 在移动或桌面设备上，您可以在应用商店中搜索“RSS”，然后选择您喜欢的免费或付费应用。
    • 如果您喜欢使用在线的阅读应用，例如Feedly，您只需要将我们的RSS源添加至您的订阅列表，即可以在其中阅读我们所有的博文。
